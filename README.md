## ⚙️ Instalação dos pacotes essenciais.

Instalar os pacotes:

```
npm install
```

Iniciar o Front-End:

```
npm run dev
```

Iniciar o Back-End:

```
npm run server
```

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   HTML e CSS.
-   JavaScript.
-   React.
-   Material UI 5.
